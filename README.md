## 🔥 Features

### 🎹 Interactive 3D Skill Showcase

- Built using **Spline** for immersive 3D interactions.
- Interactive elements visually represent my technical skills and expertise.
- Smooth hover effects and engaging user interactions.

### ✨ Modern Animations

- Powered by **GSAP** and **Framer Motion**.
- Smooth scroll animations, transitions, and component reveals.
- Enhanced user experience through creative motion design.

### 👩‍💻 Project Showcase

Features detailed information about my major projects including:

- 📈 Stock Market Prediction System
- ⚡ Electricity Billing System
- 🌦️ Weather Application
- 🎯 Career Guidance Platform
- And many more...

### 📱 Fully Responsive Design

- Optimized for desktops, tablets, and mobile devices.
- Consistent user experience across all screen sizes.

### 📬 Contact Integration

- Functional contact form powered by **Resend**.
- Easy communication for recruiters, collaborators, and potential clients.

### 🎨 Modern UI/UX

- Clean and professional design.
- Interactive components built with Shadcn UI and Tailwind CSS.
- Focus on accessibility and user experience.
